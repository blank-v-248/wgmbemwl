# Countdown-Webpage
🐎🐎🐎🐎🐎🐎🐎<br>
A single-file countdown page with no dependencies. <br>
What it does:<br>
Click the button to reveal a confetti burst and a live countdown to a given date. Every 15 seconds, a pixelated horse in sunglasses gallops across the screen. <br>Reloading the page resets everything.<br>
